# 什么是 Rust？

<img class="w-full" src="/rust-social-wide.jpg" />

Rust 是一门多范式、编译型、系统级编程语言。

它最初由 Mozilla 主导开发，现在由 Rust 项目社区开发和维护。

Rust 设计目标是提供**内存安全**、**并发友好**、**高性能**、**跨平台**等特性。

Rust 被广泛用于开发操作系统、嵌入式系统、网络服务、游戏引擎等领域 <Link to="14" title="^1" />。
